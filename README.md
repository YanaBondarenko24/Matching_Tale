# Matching Tale

Landing page for the mobile puzzle game **Matching Tale** for company **Evoplay**.

## Live Demo

https://yanabondarenko24.github.io/STP-12935/

## About

Matching Tale is a responsive promotional website for a relaxing fantasy match-3 puzzle game.

The page includes:
- Hero section
- About the game
- How to play
- Benefits
- Gallery
- Reviews
- Contact section
- Privacy Policy and Terms of Service pages

## Technologies

- HTML5
- CSS3
- JavaScript
- Swiper.js
- Vite
- GitHub Pages

## Features

- Responsive layout for mobile, tablet and desktop
- Interactive sliders
- Smooth navigation
- Mobile menu
- Game gallery
- Review cards
- Legal pages
